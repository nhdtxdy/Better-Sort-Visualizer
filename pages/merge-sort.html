Merge sort is a comparison-based sorting algorithm that works by dividing the input array into smaller sub-arrays, sorting those sub-arrays, and then merging them back together into the final sorted array.


Steps:

1. Divide the unsorted array into n sub-arrays, each containing one element (a single element array is considered sorted).

2. Repeatedly merge sub-arrays to produce new sorted sub-arrays until there is only one sub-array remaining. This will be the sorted array.

3. To merge two sub-arrays, compare the first element of each sub-array and add the smaller one to the sorted array. Repeat this process until one sub-array is empty, then add the remaining elements from the other sub-array to the sorted array.

4. Repeat step 3 until all sub-arrays have been merged.


The time complexity of merge sort is O(n log n), which is optimal for comparison-based sorting algorithms. It requires extra space proportional to the size of the input array, but this can be reduced by using an in-place merge sort algorithm.


def mergeSort(arr):
    if len(arr) > 1:
        mid = len(arr)//2
        L = arr[:mid]
        R = arr[mid:]

        mergeSort(L)
        mergeSort(R)

        i = j = k = 0

        while i < len(L) and j < len(R):
            if L[i] < R[j]:
                arr[k] = L[i]
                i += 1
            else:
                arr[k] = R[j]
                j += 1
            k += 1

        while i < len(L):
            arr[k] = L[i]
            i += 1
            k += 1

        while j < len(R):
            arr[k] = R[j]
            j += 1
            k += 1
