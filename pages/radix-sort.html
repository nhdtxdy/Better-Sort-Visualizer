Radix sort is a sorting algorithm that sorts elements by processing their individual digits. It was first developed by Herman Hollerith in the late 19th century for tabulating data on punched cards.


The algorithm sorts the elements based on their values by comparing their individual digits starting from the least significant digit to the most significant digit. Radix sort can be implemented using several different algorithms, including counting sort and bucket sort.


The basic idea behind radix sort is to sort the elements based on each digit in a fixed number of passes. For example, if the maximum number of digits in the elements is three, then radix sort will perform three passes. In each pass, it sorts the elements based on a single digit. After the final pass, the elements will be sorted based on all the digits, and the array will be sorted in ascending or descending order.


The time complexity of radix sort is O(kn), where n is the number of elements and k is the number of digits in the maximum element. Radix sort is often used for large integers or strings, where other sorting algorithms may not be as efficient.


def countingSort(arr, exp1):
    n = len(arr)
    output = [0] * (n)
    count = [0] * (10)

    for i in range(0, n):
        index = (arr[i] / exp1)
        count[int(index % 10)] += 1

    for i in range(1, 10):
        count[i] += count[i - 1]

    i = n - 1
    while i >= 0:
        index = (arr[i] / exp1)
        output[count[int(index % 10)] - 1] = arr[i]
        count[int(index % 10)] -= 1
        i -= 1

    i = 0
    for i in range(0, len(arr)):
        arr[i] = output[i]

def radixSort(arr):
    max1 = max(arr)
    exp = 1
    while max1 / exp > 0:
