Heap sort is a comparison-based sorting algorithm that uses a binary heap data structure to sort elements. It was first proposed by J.W.J. Williams in 1964 and was later refined by Robert W. Floyd in 1965 and J. Michael Steele in 1972. Heap sort is an in-place algorithm, which means it does not require any additional storage space.


The time complexity of heap sort is O(nlogn), where n is the number of elements to be sorted. This makes heap sort one of the most efficient sorting algorithms in terms of time complexity.


Steps

1. Build a max heap from the input data.
2. Find the maximum element in the heap and swap it with the last element of the heap.
3. Remove the last element (the maximum) from the heap and place it in the sorted array.
4. Repeat steps 2 and 3 until the heap is empty. 


def bubbleSort(arr):
n = len(arr)
for i in range(n-1):
    for j in range(0, n-i-1):
        if arr[j] > arr[j+1]:
            arr[j], arr[j+1] = arr[j+1], arr[j]
